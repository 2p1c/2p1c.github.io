# 背景颜色设置
## 线性渐变（Linear Gradient）
html {
  background: linear-gradient(to bottom, 
    #FF0000 0%，
    #00FF00 100%
  );
  font-size: 20px;
  color: white;
}
## 径向渐变（Radial Gradient）
html {
  background: radial-gradient(circle, green, yellow, red);
  font-size: 20px;
}
